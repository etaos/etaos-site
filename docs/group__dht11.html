Detailed Description

Device driver for the DHT11 humidity sensor.

The device driver is located in the device filesystem. Once the device file is opened, the driver needs to be setup using ioctl and the DHT11_SET_PIN register. This ioctl register takes a struct gpio_pin* as argument.
