MEM void* mm_alloc ( size_t  size)

Allocated a new memory region.

Parameters
sizeLength of the wanted region.

If a memory region which is exactly equal to or at most 4 bytes larger than size, this region is selected. If a free region is found which is 4 bytes larger than size, the region is split in two.
