Detailed Description

Run queue descriptor.

Each CPU has a run queue. A run queue is a collection of threads which are ready to receive CPU time. Each time a new thread has to be scheduled the scheduler picks a new thread from the run queue of the executing CPU.

Field Documentation

struct thread* wake_queue

Wake queue.

All threads which are put in a waiting queue are also listed on this queue. Whenever an interrupt signals the queue, the scheduler will notice by checking this queue.
