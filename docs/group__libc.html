Detailed Description

C run time library.

Note
When linking your application, pass -lc as the last library to your linker.

Complementary runtime C library. Normal C libraries (like glibc) are way to big to fit into small chips. This is a copy of a very compact c library.

Returns

The index of file will be repositioned arrording to whence as follows:

SEEK_CUR

SEEK_END
The offset is set to the size of the file plus offset bytes.
