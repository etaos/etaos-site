Detailed Description

C run time library.

Note
When linking your application, pass -lc as the last library to your linker.

Complementary runtime C library. Normal C libraries (like glibc) are way to big to fit into small chips. This is a copy of a very compact c library.

Parameters
xDestination address.
ySource address.
zNumber of bytes to copy.

Not all architectures support the 'P' version of memcpy (i.e. not all architectures can copy memory from program space to data space). If this operation is not supported, this function will redirect to the standard memcpy implementation.
