size_t lseek ( struct vfile *  file,
size_t  offset,
int  whence 
)

The lseek function repositions the offset of the given file.

Parameters
fileIndex of file will be repositioned.
offsetNumber bytes to offset file 's index.
whenceWay to offset file 's index.
Returns
The resulting offset location (file index) from the start of the file.

The index of file will be repositioned arrording to whence as follows:

SEEK_SET
The index is set to offset.

SEEK_CUR
The index is set to its current location plus offset bytes.

SEEK_END
The offset is set to the size of the file plus offset bytes.
